RightClickPasteKing

Right-click in Apple's Terminal, the way Windows and Linux do it: text selected → it copies; nothing selected → it pastes; clipboard empty → Terminal's normal menu. Nothing outside Terminal is touched.

Download the latest release

Requires macOS 13 or later. Signed and notarized; updates are delivered in-app via Sparkle. On first launch the app walks you through granting the one permission it needs (Accessibility).
